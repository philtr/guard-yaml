# Guard::Yaml

Checks your YAML syntax. That's all.

## Requirements

* [Guard](https://github.com/guard/guard) >= 1.1

## Installation

Add this line to your application's Gemfile:

    gem 'guard-yaml'

And then execute:

    $ bundle install
    $ guard init guard-yaml
